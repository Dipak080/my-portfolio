01 — About

Turning ideas into working software

I'm a Full-Stack Developer who enjoys owning a product end to end — modelling the database, designing a clean REST API, securing it with JWT, and wiring it up to a fast, intuitive frontend.

My recent flagship project, TaskOps, is a multi-tenant task-management platform with role-based permissions, company isolation, task workflows, file attachments, and real-time-style notifications — built with CodeIgniter 4 on the backend.

I care about clean architecture, reusable components, and writing code that the next developer (often future me) will thank me for.

  • 🧩 Full product ownership — backend, frontend & database
  • 🔐 Security-first APIs (JWT auth, role & permission systems)
  • 📱 Web & mobile (React + React Native)
  • 🚀 Always learning, always shipping

TaskOps — Multi-Tenant Task Management SaaS

A complete task-operations platform where multiple companies manage teams, tasks and workflows in full isolation. I built the entire backend — a secure REST API powering authentication, role-based permissions, task lifecycles, comments, checklists, attachments, notifications and end-of-day reporting.

  • 🏢 Multi-tenant architecture — isolated data per company
  • 🔐 JWT authentication + OTP-based registration
  • 🛡️ Granular RBAC — roles, pages & permissions
  • ✅ Task workflows — transitions, transfers, checklists, activity logs
  • 📎 File attachments, 🔔 notifications & 📊 EOD reports
